## Installation

### Step 1 — Panel connection

Connect your panels to the 4 MC4 input ports:

```
Panel A  ──→  Input A+ / A-
Panel B  ──→  Input B+ / B-
Panel C  ──→  Input C+ / C- (optional)
Panel D  ──→  Input D+ / D- (optional)
```

### Step 2 — AC connection

The Nano comes with a pre-wired 1.5m AC cable. Connect to a **dedicated 16A breaker** or outdoor rated socket.

!!! warning "Do not use an extension lead"
    Always connect directly to a fixed circuit or weatherproof socket.

### Step 3 — App pairing

1. Download **NanoApp** (iOS / Android)
2. Tap "Add Device" and scan the QR code on the unit
3. Connect to your home WiFi
4. Live data appears within 60 seconds

## Troubleshooting

| Symptom | Likely Cause | Fix |
|---|---|---|
| No output | AC not connected | Check breaker and socket |
| Low output | Shading on panels | Check for obstructions |
| App not connecting | Bluetooth out of range | Move phone within 5m during setup |
